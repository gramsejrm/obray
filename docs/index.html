Introduction

The obray PHP framework was developed as a minimalistic approach to building web based applications. The basic tasks the framework is attempting to accomplish are the following:

  1. Receive URI (universal resource identifier) and translate that into a class and function to be executed by PHP
  2. Enforce permissions on each of those URIs
  3. Provide basic access to the database and handle the database connection

Based on the approach above the framework provides a simple groundwork for more complex tasks to be accomplished.
